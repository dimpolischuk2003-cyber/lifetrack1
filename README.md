# LifeTrack 🎯💳

Трекер звичок та фінансів — PWA додаток.

---

## 🚀 Як задеплоїти на Vercel (покроково)

### Крок 1: Створи GitHub репозиторій

1. Зайди на [github.com](https://github.com) (зареєструйся, якщо ще немає акаунту)
2. Натисни **"+"** → **"New repository"**
3. Назви його `lifetrack`
4. Натисни **"Create repository"**

### Крок 2: Завантаж файли проекту на GitHub

**Варіант A — через браузер (найпростіший):**
1. На сторінці нового репозиторію натисни **"uploading an existing file"**
2. Перетягни ВСІ файли та папки з цього проекту у вікно браузера
3. Натисни **"Commit changes"**

**Варіант B — через термінал:**
```bash
cd lifetrack-app
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ТВІЙ_ЮЗЕРНЕЙМ/lifetrack.git
git push -u origin main
```

### Крок 3: Підключи Vercel

1. Зайди на [vercel.com](https://vercel.com) і натисни **"Sign Up"** → увійди через GitHub
2. Натисни **"Add New..."** → **"Project"**
3. Знайди репозиторій `lifetrack` і натисни **"Import"**
4. Все вже налаштовано — просто натисни **"Deploy"**
5. Через ~1 хвилину отримаєш посилання типу `lifetrack-xxx.vercel.app`

### Крок 4: Встанови на телефон

**iPhone (Safari):**
1. Відкрий посилання в Safari
2. Натисни іконку "Поділитися" (квадрат зі стрілкою)
3. Скрольни вниз → **"На Початковий екран"**
4. Натисни **"Додати"**

**Android (Chrome):**
1. Відкрий посилання в Chrome
2. Натисни три крапки (⋮) зверху
3. Натисни **"Додати на головний екран"** або **"Встановити додаток"**
